## 💫 About
A C# console application that demonstrates how to build a flexible input data validation system by combining custom attributes, reflection and regular expressions (Regex). The project is tightly organised according to the MVC (Model-View-Controller) pattern.

## 📂 Directory Structure

```plaintext
StaffValidationMVC/
│
├── Attributes/
│   ├── EmailCheckAttribute.cs
│   └── PhoneCheckAttribute.cs
│
├── Controllers/
│   └── StaffController.cs
│
├── Dto/
│   └── StaffValidatorDto.cs
│
├── Models/
│   └── Staff.cs
│
├── Validation/
│   └── StaffValidator.cs
│
├── Views/
│   └── StaffView.cs
│
└── Program.cs
```
